In this repository over time several board designs will be added that allow the creating of devices that use MIDI and digital audio.

The boards use the following components sourced from Aliexpress:

Stereo mini jack PJ-327A  https://www.aliexpress.com/item/1005005394455266.html

Mono 6.35mm jack          https://www.aliexpress.com/item/1005006612950070.html

5 pin DIN connector       https://www.aliexpress.com/item/1005006314983600.html

SN74LVC126                https://www.aliexpress.com/item/1005006250704206.html

PCM1808 ADC module        https://www.aliexpress.com/item/1005008885079639.html

PCM5102 DAC module        https://www.aliexpress.com/item/1005006104038963.html

PCM5102 DAC module        https://www.aliexpress.com/item/1005006829178513.html


For the MIDI in part the schematics show the 6N139 optocoupler, but the 6N137 can also be used, for which the pullup R2 can be increased to 2k2 or higher.
